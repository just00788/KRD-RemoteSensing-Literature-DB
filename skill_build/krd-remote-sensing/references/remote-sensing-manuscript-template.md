# Remote Sensing Manuscript Template Reference

## When to use this reference

Use this reference when generating a manuscript outline or converting project results into a Remote Sensing-ready article structure.

## Recommended Main Sections

1. Introduction.
2. Materials and Methods.
3. Results.
4. Discussion.
5. Conclusions.

## Recommended Subsections

### Materials and Methods

- Study areas.
- Remote-sensing and DEM data.
- Reference samples and classification system.
- FBR inversion.
- DeepLabV3+, FCN, and U-Net.
- DEM fusion and ablation.
- Cross-region transfer validation.
- Accuracy assessment.

### Results

- FBR inversion.
- Thresholded KRD grade mapping.
- Model comparison.
- DEM ablation.
- Cross-region transfer.
- GF-7 stereo DEM local enhancement.
- Error and class-level analysis.

### Discussion

- Model advantages and mechanisms.
- Physical meaning of FBR.
- DEM/topographic effects.
- Transferability and local enhancement.
- Comparison with prior studies.
- Limitations and future work.

## Layout Targets

- Abstract: 180-250 words.
- Introduction: 5-6 paragraphs.
- Methods: 6-8 subsections.
- Results: 6-8 subsections.
- Discussion: 5-6 subsections.
- Figures: 8-10 main figures if the study is complete.
- Tables: 4-6 main tables if the study is complete.
