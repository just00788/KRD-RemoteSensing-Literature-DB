# Remote Sensing Journal Writing

## When to use this reference

Use this reference when drafting, restructuring, or auditing a manuscript for Remote Sensing style.

## Standard Structure

Use an IMRaD-style structure:

1. Introduction.
2. Materials and Methods.
3. Results.
4. Discussion.
5. Conclusions.
6. Back matter: Data Availability, Author Contributions, Funding, Conflicts of Interest, and Supplementary Materials when needed.

## Abstract Pattern

Use:

1. Background.
2. Problem or gap.
3. Data.
4. Method.
5. Key validated result.
6. Significance.

Keep metric types separate. Do not mix continuous FBR metrics with classification metrics.

## Introduction Pattern

Use 5-6 paragraphs:

1. KRD ecological and monitoring context.
2. FBR and remote sensing relevance.
3. Prior LSMM, SMA, MESMA, feature-space, and ML work.
4. Deep learning, segmentation, and terrain constraints.
5. Gap.
6. Objectives and contributions.

## Methods Pattern

Make methods reproducible:

- Study areas.
- Data.
- Preprocessing.
- Reference samples.
- FBR/LSMM workflow.
- DeepLabV3+ and baselines.
- DEM fusion and ablation.
- Transfer validation.
- Metrics.

## Results Pattern

Organize around evidence objects:

- Maps.
- Metric tables.
- Comparison tables.
- Ablation tables.
- Confusion matrices.
- Error or uncertainty maps.

## Discussion Pattern

Interpret mechanisms and limitations. Do not repeat the Results section.
