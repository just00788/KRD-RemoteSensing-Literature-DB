# Literature Map

## When to use this reference

Use this reference when positioning the manuscript in the KRD literature or deciding which prior-work clusters support a claim.

## Evidence Clusters

- KRD classification and information extraction.
- FBR, bare-rock fraction, and exposed bedrock fraction.
- LSMM, SMA, MESMA, and feature-space methods.
- RF, SVM, and other machine-learning classifiers.
- DeepLabV3+, FCN, U-Net, and semantic segmentation.
- DEM and terrain-factor integration.
- GEE and time-series monitoring.
- Remote Sensing journal article structure and style.

## How to Use Literature Evidence

- Use scientific literature to support problem framing, method choice, and comparison logic.
- Use Remote Sensing style papers to guide structure, not to copy prose.
- Keep paper-specific claims tied to verified metadata or project paper cards.
- Mark uncertain metadata or OCR-corrupted titles as `needs verification`.

## Gaps to Preserve

- Cross-region validation remains a key gap.
- DEM ablation should be explicit.
- GF-7 stereo DEM should be positioned as local enhancement.
- Continuous FBR and categorical KRD levels must be separated.
- Class-level metrics should be reported for classification and segmentation.
