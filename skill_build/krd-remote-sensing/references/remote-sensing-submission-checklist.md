# Remote Sensing Submission Checklist

## When to use this reference

Use this reference for pre-submission checks, final manuscript audits, and before drafting a cover letter or response package.

## Scientific Checks

- FBR is treated as continuous unless thresholded.
- LSMM/SMA/MESMA outputs are not described as classification maps.
- FBR thresholds are defined before class comparisons.
- Continuous metrics and classification metrics are separated.
- Spatial block split or transfer validation is stated.
- Open-source DEM and GF-7 stereo DEM roles are separated.
- DeepLabV3+, FCN, and U-Net roles are clear.

## Manuscript Checks

- Abstract follows background-gap-data-method-result-significance.
- Introduction states a precise gap.
- Methods are reproducible.
- Results are organized around figures and tables.
- Discussion interprets mechanisms and limitations.
- Conclusions are concise and restrained.

## Figure and Table Checks

- Every figure/table is cited in text.
- Captions describe what is shown.
- Study area map and workflow figure are included if appropriate.
- Metrics table includes the correct metric family.
- Ablation and transfer results are separated.

## Back Matter

Prepare:

- Data Availability.
- Author Contributions.
- Funding.
- Conflicts of Interest.
- Supplementary Materials when needed.
